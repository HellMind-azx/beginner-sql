# beginner-sql# SQL Интенсив — PostgreSQL с нуля

Практический 5-дневный курс для Django-разработчиков, которые хотят уверенно писать SQL без ORM.

## О курсе

Курс построен по принципу «теория → практика → задание». Каждый следующий день открывается только после того, как ты решишь контрольное задание предыдущего. Все объяснения сопровождаются аналогиями из Django ORM — чтобы новые концепции ложились на уже знакомую базу.

Фокус — PostgreSQL. Никакой воды, только то, что нужно для написания реальной логики приложений.

## Программа

**День 1 — DDL: проектирование схемы**
Создание таблиц (`CREATE TABLE`), типы данных PostgreSQL, ограничения (`NOT NULL`, `UNIQUE`, `DEFAULT`, `CHECK`), первичные ключи. Задание: спроектировать схему данных для блога.

**День 2 — DML: работа с данными**
Вставка (`INSERT`), обновление (`UPDATE`), удаление (`DELETE`), выборка (`SELECT`) и фильтрация (`WHERE`, `LIKE`, `IN`, `BETWEEN`). Задание: наполнить блог данными и написать специфические выборки.

**День 3 — Связи между таблицами**
Внешние ключи (`FOREIGN KEY`), поведение при удалении (`CASCADE`, `SET NULL`, `RESTRICT`), связи один-к-одному и один-ко-многим. Задание: связать пользователей со статьями и комментариями.

**День 4 — JOIN: объединение таблиц**
`INNER JOIN` и `LEFT JOIN`, разница между ними и когда что использовать, объединение трёх и более таблиц. Задание: вывести все статьи с именами авторов одним запросом.

**День 5 — Агрегация и группировка**
Агрегатные функции (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`), группировка (`GROUP BY`), фильтрация групп (`HAVING`), разница между `WHERE` и `HAVING`. Задание: посчитать статистику по авторам.

## Как запустить

Открой файл `sql_course.html` в любом браузере. Установка не требуется.

Для выполнения SQL-запросов понадобится PostgreSQL. Подойдёт локальная установка с `psql` или графический клиент — например, DBeaver или TablePlus.

## Требования

- Браузер (Chrome, Firefox, Safari, Edge)
- PostgreSQL 13+
- Базовое знакомство с Python и Django (не обязательно, но помогает)

## Прогресс

Курс отслеживает выполненные задания и сохраняет прогресс в браузере. Каждый день разблокируется автоматически после проверки задания.
